Hướng dẫn kết nối Unity với VS Code

Các bạn nhớ xem kĩ hướng dẫn và hình ảnh rồi mới làm theo nha

Bước 1: Các bạn tạo 1 project 2D mới nha theo các bước như sau

Trước khi bấm “Create Project” nhớ set location cho project, không là không xác định được project nằm ở đâu cả. Bấm xong nó build project hơi lâu 1 tí nên kiên nhẫn đợi bạn nhé

**Ghi chú: Project 2D được build nhanh hay chậm tùy thuộc vào sức mạnh của CPU, RAM, tốc độ đọc của ổ cứng bạn nhé. (Thường sẽ mất 1 khoảng thời gian khá lâu)

Bước 4: Chúng ta đã thấy được giao diện ban đầu của Unity. Để được coi là setup hoàn tất thì chúng ta còn một vài bước nữa để connect giữa Unity và VS Code. Các bạn làm theo hướng dẫn nhé

Bước 4: Nếu bạn đã có setup VS Code xong rồi thì chỗ này bấm dropdown xuống sẽ có tên, nếu như setup xong mà vẫn chưa xong thì bạn có thể bấm browse từ danh mục của dropdown list

Bước 5: các bạn kiếm vào mục có đuôi .exe nhé còn tên thì mình không rõ có khi nó để khác.các bạn nhìn theo tên location mình đã khoanh tròn đỏ nha, có khi location khác thì các bạn kiếm bằng cách sau: “Bấm chuột phải vào biểu tượng của Visual Studio trên Desktop -> chọn mục Properties và kiếm location của file exe ở Window Explorer”. Sau đó các bạn bấm Open là xong

Như vậy tới được bước này là cơ bản bạn đã hoàn tất rồi đó

Chắc chắn sẽ không tránh được việc có lỗi trong quá trình cài. Nếu vậy thì còn ngại gì mà không ẤN VÀO ĐÂY để email cho chúng mình bạn nhỉ!
