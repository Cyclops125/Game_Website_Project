Một số bài học

!. Vòng lặp for nhằm tạo ra một cấu trúc, thi hành khối lệnh nào đó một một số lần chỉ định như sau:

  • Điều kiện 1: Đầu tiên nó thi hành câu lệnh tại khởi_tạo, thường chỗ này ta khởi tạo bộ đếm vòng lặp như int i = 1; j = 10;
  • Điều kiện 2: Kiểm tra biểu thức logic điều_kiện, nếu true thì bắt đầu thi hành khối lệnh { ... } sau đó chạy tiếp điều kiện 3, nếu false thì thoát lặp
  • Điều kiện 3: Thi hành biểu thức cập_nhật, chỗ này ta thường dùng để tăng - giảm bộ đếm như i++;, thi hành xong thì quay lại điệu kiện 2

2. Câu lệnh while() này đầu tiên kiểm tra điều_kiện nếu bằng true thi hành khối lệnh nếu false thoát vòng lặp, thi hành xong lại kiểm tra lại điều_kiện và nếu vẫn bằng true lại thi hành lại khối lệnh ... cứ thế cho đến khi điều_kiện là false

Cần lưu ý, viết vòng lặp while cần set điệu kiện lặp hợp lý, nếu không sẽ lỗi hoặc lặp vô hạn (infinitive loop).

3. Vòng lặp do-while này khác while ở chỗ, nó thi hành khối lệnh trước, sau đó mới kiểm tra điều_kiện, kiểm tra thấy điều_kiện vẫn true thì lặp tiếp.
