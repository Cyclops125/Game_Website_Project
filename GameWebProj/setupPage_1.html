Hướng dẫn cài đặt Unity

Các bạn nhớ xem kĩ hướng dẫn và hình ảnh setup nhá

Bước 1: Các bạn truy cập theo link: https://unity3d.com/get-unity/download

Bước 2: Web sẽ tự động tải về file setup có tên là UnityHubSetup.exe

Bước 3: Các bạn bấm vào nút "Download Unity Hub" và bấm mở file UnityHubSetup.exe để run setup nha

Bước 4: Sau khi tải xong các bạn tìm file setup có tên như trên và bấm vào để setup

Bước 5: Bạn chọn folder chứa và bấm Agree để tiếp tục Install nha!

Bước 6: Các bạn nhớ trước khi ấn “finish” thì nhớ tick vào ô “Run Unity Hub” để chạy chương trình sau khi setup xong để check có lỗi không nha

Bước 7: Sau khi Unity Hub được mở, các bạn bấm vào mục “Install” nếu chưa có gì hết thì tiếp tục bấm vào nút “Install Editor” và install version nào cũng được

Bước 8:Tiếp theo nó sẽ lên 1 cái bảo như vầy, bạn bỏ chọn những tích xanh nha rồi bấm “install”.
